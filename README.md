## Open Source: BaseRoute

[BaseRoute](https://github.com/jaofrod/baseroute) is my open-source Base DeFi automation project.

It is a deterministic yield routing bot for Base that reads live on-chain data, compares USDC supply opportunities across protocols, applies explicit safety rules, and can run in paper
mode before any live execution.

The project is intentionally public and educational:

- deterministic strategy engine;
- Base mainnet data reading;
- Aave V3 and Compound III integration;
- safety firewall before execution;
- paper trading mode;
- optional AI analysis as a cold path;
- local dashboard and runtime visibility.

BaseRoute is where I am publicly building, documenting, and improving my Web3 engineering practice.

## Moonrad

At [Moonrad](https://moonrad.com), we are building **Voidnaut**, an on-chain risk intelligence system for Base.

Voidnaut is designed for teams that need to monitor contracts, wallets, deployers, and suspicious activity through an API-first product surface.

Current focus:

- real-time Base activity monitoring;
- deterministic contract risk analysis;
- alert pipelines and webhook delivery;
- B2B API infrastructure;
- security-focused product architecture.

Public architecture repo:

- [voidnaut-core-architecture](https://github.com/jaofrod/voidnaut-core-architecture)

Voidnaut is **closed-source**. The public repository documents architecture, design boundaries, and technical decisions without exposing proprietary detection logic or private
implementation.
